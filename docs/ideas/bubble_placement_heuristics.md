# Bubble Placement Heuristics

吹き出し配置を最適化問題として扱う前に、まず「良い置き方」を言語化しておく。
このメモは、実装前の判断基準を整理するためのもの。

## 基本方針

- 吹き出しは基本的に右上から左下へ読む流れを作る
- 画像全体として、視線が右上から左下へ自然に落ちる並びを優先する
- 吹き出しは人物や表情と無関係な空白に機械的に置くのではなく、話者の存在に結びついた位置に置く

## 基本の配置順

単一人物が画面中央付近に立っている構図では、まず次のどちらかを基本順序として考える。

### 1. 通常の N 順

- 右上
- 右下
- 左上
- 左下

これは、右列を上から下に使ってから左列へ移る考え方。

### 2. 90度回転した N 順

- 右上
- 左上
- 右下
- 左下

これは、上段を右から左へ使ってから下段へ移る考え方。

## どちらを選ぶか

- 基本は「右上から左下へ向かう」ことを守る
- ただし、人物の姿勢、顔の向き、体の傾き、画面内での立ち位置によって自然な順序は変わる
- 人物が中央に立つ縦長構図では通常の N 順がはまりやすい
- 上半身が大きく、上側に空きがある構図では 90度回転した N 順が自然なことがある
- 会話の密度が高いときは、単純な順序よりも吹き出し同士のまとまりを優先する

## 顔との関係

- 吹き出しは顔の近くに置きたい
- ただし顔に近すぎて、表情を読む邪魔になる位置は避けたい
- 顔から適度に近く、少し外側へ逃がした位置がよい
- 顔の真上や斜め上は有力候補になりやすい
- 顔のすぐ横でも、視線や髪や輪郭を強く潰す位置は避けたい

実装上は次のように扱える。

- 顔との距離が遠すぎると減点
- 顔との距離が近すぎても減点
- 顔から一定距離帯に入る候補を優遇する

## 画像の枠との関係

- 吹き出しは画像の枠に近すぎないほうがよい
- ただし、外周寄りに置くこと自体は悪くない
- 問題なのは「枠に貼り付いた感じ」や「窮屈さ」が出ること

実装上は次のように扱える。

- 画面端からの最小マージンを設ける
- マージン違反は大きく減点する
- 外周寄りだが十分な余白がある配置は許容する

## 人物への重なり

人物全体に一切かぶせない、ではなく、重要部位への重なりを強く避ける。

### 強く避けたい部位

- 顔
- 胸
- 股間

### できれば避けたいが、多少は許容できる部位

- 肩
- 腕
- 脚
- 胴体の外周

つまり、人物マスク全体への overlap は連続量として扱い、部位ごとに重みを変えるのがよい。

実装上の優先度はおおむね次の通り。

- 顔 overlap: 失格
- 胸 overlap: ほぼ失格、または非常に大きい減点
- 股間 overlap: ほぼ失格、または非常に大きい減点
- その他の人物 overlap: 小さければ許容、面積に応じて減点

## 会話の連続性

- 続いている会話なら、吹き出し同士はあまり離したくない
- 同一話者の連続発話やテンポの速い掛け合いでは、空間的なまとまりが重要
- 逆に、話題転換や間がある発話なら、少し離れていてもよい

実装上は次のように扱える。

- 連続する吹き出し間の距離にペナルティを入れる
- 特に前後の bubble が会話ペアである場合は、距離ペナルティを強める
- ただし、無理に近づけて顔や主要部位を隠すなら、そちらを優先して避ける

## 望ましい見え方

- 視線誘導が迷わない
- 顔の近くにあるので、誰の発話か理解しやすい
- 枠際に押し込まれた感じがない
- 人物の大事な情報を潰していない
- 連続する会話が空間的にもまとまって見える

## 最適化に落とすときの初期ルール

最初の PoC では、次のようなコストで十分。

- 読み順違反: 大きい減点
- 顔 overlap: 失格
- 胸 overlap: 非常に大きい減点
- 股間 overlap: 非常に大きい減点
- 画面外にはみ出す: 失格
- 端マージン不足: 減点
- 顔から遠すぎる: 減点
- 顔に近すぎる: 減点
- 連続 bubble 間の距離が大きい: 減点
- その他人物領域との overlap: 面積に応じて減点

## まだ決めきっていない点

- 胸マスク、股間マスクをどう用意するか
- 複数人物がいるとき、どの顔に寄せるか
- 左右どちらに寄せるのが自然かを、顔向きや視線からどこまで判断するか
- 通常の N 順と 90度回転 N 順を、どの条件で切り替えるか
- しっぽを導入した場合に、bubble 本体位置と話者対応をどう再評価するか

## まとめ

吹き出し配置は、「空いている場所に置く問題」ではなく、
「読み順」「顔との関係」「主要部位の保護」「会話のまとまり」を同時に満たす配置問題として扱う。

特に単一人物構図では、右上から左下へ向かう流れを保ちつつ、
顔の近く、枠に近すぎず、主要部位を避け、連続発話は近くに置く、という方針を基本にする。
